自分用のメモなので, 説明が不足していることがあります

基本的に, TypeScript と Go (最近はもっぱら Go です)で書いています

import { useData } from "nextra/data";
import { getRating } from "api/rating";

export const getStaticProps = async ({ params }) => {
  const data = await getRating();
  return {
    props: { ssg: { ...data } },
    revalidate: 60 * 60 * 24,
  };
};

export const Rating = () => {
  const { rating, colorCode } = useData();
  return <strong style={{ color: colorCode }}>{rating}</strong>;
};

export const HighestRating = () => {
  const { highestRating, highestColorCode } = useData();
  return <strong style={{ color: highestColorCode }}>{highestRating}</strong>;
};

現在のレートは以下の通りです

Rating: <Rating />  
Rating 最高値: <HighestRating />
