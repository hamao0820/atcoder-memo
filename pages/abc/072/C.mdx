## 解説

[問題](https://atcoder.jp/contests/abc072/tasks/arc082_a)

$[a_i - 1, a_i + 1]$ の区間を考え、その区間が重なる回数が最大となる $X$ を求める問題。

この問題は、いもす法を用いて解くことができる。

出力する値は、いもす法の累積和の最大値となるので、並行移動しても問題ない。配列として管理するために、$a_i$　を $+1$ 、$a_i + 3$ を $-1$ としていもす法を適用する。

## 提出コード

```go
func main() {
	readInt()
	A := readInts()
	imos := make([]int, 1000003)
	for _, v := range A {
		imos[v]++
		imos[v+3]--
	}
	for i := 1; i < 1000002; i++ {
		imos[i] += imos[i-1]
	}
	writeLine(max(imos...))
}
```
